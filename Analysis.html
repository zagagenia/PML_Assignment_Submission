Abstract

We use the random forest method to estimate features for the Human Activity Recognition data set from Groupware.

Data

The data is taken from the Human Activity Recognition programme at Groupware. We believe that Random Forest Model that we train will predict testing data by an Accuracy of 95%. We estimate tree size and depth as follows; tree size is a multiple of depth. According to Manual On Setting Up, Using, And Understanding Random Forests V3.1 1. A large number of trees, 10000, potentially make for stable variable importances, thus better models.

Results

Confusion Matricies Compairson of Training and Testing Sets

We examine model training and prediction and contrast results using the "Classe Variable cited in the above data set. A detailed explaination of the R-Code and how it works can be found of the .Rmd file in this repo. We discuss compare the results from the training and testing sets using confusion matricies and statistics. The definitions for these terms and statistics can be found in Package Caret, August 15, 2014 Version 6.0-34. Accuracy of training compared to testing is 1.0 to 0.9949 indicating training for test prediction are accurate.Sensitivity are 1.0 to 0.9993 meaning that the model predictions predict "Classe" membership nearly 100% of the time. Specificity are 1.0 to 0.9986 that the model predictions predict "Non-Classe" membership nearly 100% of the time. In conclusion, the model is well-trained with the right tree size based on the training and test set provided.
